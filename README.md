## 📁 Repository Structure

Problems are grouped by topic. Each folder contains the solution file named after the problem's contest ID and index (e.g. `1873A.cpp`), with a short comment at the top describing the approach.

```
CodeForces-Solutions/
├── DP/
├── Graphs/
├── Greedy/
├── Math/
├── Strings/
├── Binary-Search/
├── Implementation/
└── Misc/
```

## 🚀 How to Run

**C / C++**
```bash
g++ -std=c++17 -O2 -o sol solution.cpp
./sol < input.txt
```

**Python**
```bash
python3 solution.py < input.txt
```
